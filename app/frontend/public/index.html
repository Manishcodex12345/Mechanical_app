Documents / User Manual

User Manual (Requester)

  1. Go to Requester Portal from the left navigation.
  2. Fill invoice number, material details, weight, cost, destination, ship date, and priority.
  3. Click Submit Request.
  4. Wait for in-app confirmation: “Request submitted and HR admin notified.”
  5. Use priority correctly: High only for urgent shipments.

User Manual (HR Admin)

  1. Open HR Admin Dashboard from the left navigation.
  2. View all pending requests sorted by priority and age.
  3. Use High/Medium/Low filters to quickly process urgent items.
  4. Review invoice, weight, cost, destination, and ship date in each card.
  5. Prioritize high-priority or overdue requests first.
