      User Manual (Requester)

      1. Open Create Request and enter invoice + material details.
      2. Set priority correctly (High only for urgent or customs-sensitive shipments).
      3. Submit and track updates in My Requests.
      4. If clarification is requested, update details and resubmit quickly.

      User Manual (HR Admin)

      1. Open HR Admin Queue and filter by priority.
      2. Review details: weight, cost, destination, and ship date.
      3. Take action: Approve / Reject / Clarification Needed.
      4. Use comments for rejection or clarification to improve turnaround.
